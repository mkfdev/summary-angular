컴포넌트 간 커뮤니케이션

  • 루트 컴포넌트 : app-http-todo
  • 자식 컴포넌트 : app-todo-input / app-todo-select / app-todo-list / app-todo-footer
  • @Input : 부모 컴포넌트 -> 자식 컴포넌트
  • @Output : 자식 컴포넌트 -> 부모 컴포넌트

1. root 템플릿(app-htto-todo)에서 각 컴포넌트 요소에 이벤트 바인딩 / 데이터 바인딩

2. 해당하는 자식 컴포넌트에서 root 컴포넌트로 데이터 상태와 이벤트를 전달한다.(현재는 인라인 형식으로 html이 작성됨)

3. @Output으로 emit한 이벤트 함수를 가지고 root(부모) component에서 로직을 구현한다.
나머지 자식 컴포넌트도 @Input/@Output을 통해 이벤트와 데이터를 주고 받는다.
